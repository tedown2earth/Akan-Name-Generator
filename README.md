# Akan-Name-Generator
 
 ## Description
The Akan Name Generator is a simple web application that calculates the day of the week you were born and gives you your Akan name based on your gender.  
Akan names come from Ghanaian culture, where children are named according to the day of the week they were born.

## Setup Instructions
1. Clone the repository

git clone https://github.com/tedown2earth/Akan-Name-Generator.git

2. Navigate into the project folder.

cd Akan-Name-Generator

3. Open index.html in your browser.

4. Enter your date of birth and gender. Thereafter click the "Get Akan Name" button to see your Akan Name.

## Behaviour-Driven Development (BDD)

. Given a user enters a valid date and selects their gender and when they submit the form, the generator displays the day of the week and their Akan name.

. When a user enters an invalid date or leaves the gender space blank and when they submit the form, the application shows an alert asking for valid input.
